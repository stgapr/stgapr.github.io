日文书籍排版、转换、阅读工具

2019-08-31 11:49:02 | 笔记

日文书有一些稀奇古怪的排版、字符集问题，包括（但不限于）青空文库里那些带标记的TXT/XHTML，所以我就收集整理了一些相关检索/转换/软件。

下面部分软件是书籍爱好志愿者开发的，未经我测试，有的可能很久没更新了，仅供参考。太普通的如Adobe PDF reader就不列在内。

元指南

  • aozora bunko FAQ

  • 与青空文库有关的工具列表 - Github

    包括一些文字转换工具、校对工具：txt2xhtml，ruby parser，emacs lisp脚本，新旧字体转换的“校閲君”，Emacs插件，OCR软件NHocr和日本软件厂商的商用阅读器等，下面不再重复列描述。

  • 青空文庫マニュアル

资源与检索

  • 之前整理过一批可以阅读或查询日文电子书籍的地方，此处不再赘述。

  • 青空文库相关资源检索

    • 青空文库的Github同步备份

    • うわづら文庫 - 图像版的青空文库

    • Aozora WING
      青空文庫で公開されている作家・作品情報と全作品テキストを一つにまとめたEPWINGデータ

    • 無名作家の青空 - 著作権が消滅した作家の一覧、text file editor、底本の探し方など

格式转换

青空文库提供HTML和zip压缩后的TXT格式，均为IS X 0208／ShiftJIS编码。HTML的排版比较好（用ruby标签实现了振假名），但有时候我们需要转换成别的格式，并且考虑到振假名和文字方向的问题。

  • Aozora Epub3
    把青空文庫的带标记txt文档转换成epub3（zip压缩）的工具。该工具同时也支持一大票日本国内网络小说站点的格式。因为对TXT文档的各种手工标记做了排版处理，转换效果很好。

  • 青空キンドル Beta
    青空文庫 のテキストを、Kindle で快適に読めるような PDF に変換するサイト。
    是一个在线转换工具，支持各种尺寸的Kindle。

  • 青空文庫を読もう！
    借助TeX，把带标记的青空文库txt文件转换成PDF。另外见青空スタイル、aozora tex、aozora blog 2008/6/22

  • Aozora XMDF
    青空文庫の注記入りテキストファイルを、XMDFビルダーのテキストフローの独自タグのテキストに変換するツールです。

  • Calibre - 多格式电子书管理、转换和阅读工具
    通用的文献管理工具，基本上什么格式都能转换。在国内很有名气，但我不是很确定它是否支持日文排版特性，因为开源工具不一定比专用工具处理得好。

    阅读和编辑

  • えあ草紙
    官网：えあ草紙工房 ，一个在线的、支持TXT/ZIP TXT/HTML(URL) 日文文档阅读js插件，支持嵌入其他网站。有多个衍生版本:

    • 和青空文库集成的えあ草紙·青空図書館
    • Windows/Mac/Linux桌面版：AIR草紙

  • 青空文库书籍有声书计划

  • Tobira/扉/とびら
    Windows平台免费软件，支持ruby（振假名），支持Shift_JIS/EUC/UTF8/UTF16，文字抗锯齿，自动换行，横向/竖向切换，链接跳转，更改字体和背景，缩进。最后一次维护在2006年，支持Windows 98/XP/2000，你还是退群吧。

  • Skybook

  • Mery
    带代码高亮的文本编辑器，类似notepad++

  • WZ系列编辑器
    包括“Text Editor WZ EDITOR 10”和“Writing Editor WZ Writing Editor 2”等（付费软件）

  • aozora editor 或 aozora editor

  • O’s editor 2
    支持大文件编辑，所见即所得，央视切换、丰富的打印样式，支持垂直书写、代码高亮、缩进排版等功能的编辑器。

  • Microsoft Edge
    Windows10 自带的浏览器，可以拿来看epub。不支持Windows10以下系统。 据说未来版本会移除对epub的支持。不是很建议用。

  • Sumatra PDF
    Windows平台软件，支持PDF epub mobi xps djvu chm cbz cbr
